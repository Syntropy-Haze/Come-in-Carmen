## Pattern Analysis & Interpretation

### 1. Complementary Response Patterns

The most striking finding is the **opposite response patterns** between sensors:

- **Condition A:** Sensor A responds positively in Run 1 (+17%) but minimally/negatively in Run 2 (-4%), while Sensor B shows the opposite pattern
- **Condition B:** Strong positive response in Sensor A for both runs, variable response in Sensor B
- **Condition AB:** Consistent negative response in Sensor A, positive or neutral in Sensor B

This suggests the two light sources may be **coupled** or **complementary** in some way, where enhancement of one corresponds to suppression or different behavior in the other.

### 2. Order Effects

Despite reversing the order of conditions between runs:
- Sensor A showed stronger response to Condition B in both runs
- The combined condition (AB) consistently decreased Sensor A luminosity
- Effect magnitudes varied but directional patterns were partially preserved

### 3. Statistical Reliability

All observed effects showed **high statistical significance** (p < 0.001), with effect sizes ranging from small to large (Cohen's d from 0.18 to 0.91). This indicates the observed patterns are unlikely to be due to random variation.

### 4. Temporal Dynamics

The weak within-condition correlations but stronger overall correlations (especially in Run 2) suggest:
- Independent short-term fluctuations between sensors
- Possible longer-term coupling or shared environmental factors
- Time lags between sensors indicate potential causal relationships
